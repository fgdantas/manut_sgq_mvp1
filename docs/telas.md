# Telas do MVP 1

## 1. Dashboard
### Objetivo
Mostrar uma visão geral do sistema.

### Informações exibidas
- quantidade de preventivas vencidas
- manutenções programadas para hoje
- máquinas/equipamentos em manutenção
- ordens abertas

---

## 2. Máquinas e Equipamentos
### Objetivo
Cadastrar e consultar os ativos do sistema.

### Campos principais
- código
- nome
- tipo
- setor
- status

---

## 3. Cadastro de Preventiva
### Objetivo
Cadastrar uma manutenção preventiva.

### Campos principais
- equipamento
- atividade
- frequência
- responsável
- próxima execução

---

## 4. Lista de Preventivas
### Objetivo
Acompanhar as preventivas cadastradas.

### Informações exibidas
- equipamento
- atividade
- próxima execução
- responsável
- status

---

## 5. Manutenção Corretiva
### Objetivo
Registrar uma manutenção corretiva.

### Campos principais
- equipamento
- data
- problema
- ação realizada
- responsável
- tempo parado

---

## 6. Histórico
### Objetivo
Consultar o histórico de manutenções.

### Filtros
- equipamento
- período
- tipo de manutenção
