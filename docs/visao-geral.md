# Visão Geral do Projeto

## Nome do projeto
Sistema de Controle de Manutenção - MVP 1

## Problema
A empresa precisa controlar de forma organizada as manutenções de máquinas e equipamentos, evitando falhas, atrasos e perda de histórico.

## Solução proposta
Criar um sistema simples para cadastrar ativos, planejar manutenções preventivas, registrar manutenções corretivas e consultar o histórico.

## Foco da primeira versão
A primeira versão será enxuta e focada no essencial:
- cadastro de ativos
- manutenção preventiva
- manutenção corretiva
- histórico

## Benefícios esperados
- maior organização
- melhor rastreabilidade
- redução de esquecimentos de manutenção
- apoio ao controle operacional
