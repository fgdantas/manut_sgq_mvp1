# Sistema de Controle de Manutenção - MVP 1

## Objetivo
Criar a primeira versão de um sistema simples para controlar manutenções de máquinas e equipamentos.

## Escopo do MVP 1
A primeira versão terá apenas as funções essenciais para iniciar o controle operacional e gerar rastreabilidade.

## Funcionalidades do MVP 1
- Cadastro de máquinas e equipamentos
- Cadastro de manutenção preventiva
- Lista de manutenções preventivas
- Registro de manutenção corretiva
- Histórico de manutenções
- Dashboard inicial simples

## Telas do MVP 1
1. Dashboard
2. Máquinas e Equipamentos
3. Cadastro de Preventiva
4. Lista de Preventivas
5. Manutenção Corretiva
6. Histórico

## Estrutura do projeto
- docs/: documentação do sistema
- frontend/: layout inicial das telas

## Status
Projeto em fase inicial de estruturação.
